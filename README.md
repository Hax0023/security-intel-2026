- memory-binary/ — BOF, UAF, RCE
- infra-cloud/ — AWS/K8s/GCP, privesc
- supply-chain/ — npm/PyPI, CI/CD

- business-logic/ — Race conditions, logic flaws
- hardware-firmware/ — CPU, UEFI, embedded

- learning-notes/ — Your personal notes
